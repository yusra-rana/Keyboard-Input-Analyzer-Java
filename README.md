# ⌨️ Keyboard Input Analyzer

A Java-based application that analyzes text entered by the 
user and provides detailed character-by-character breakdown.


## 📋 Features
- 📝 **Total Characters** — Counts all characters in input
- 📊 **Total Words** — Counts total words entered
- 🔤 **Vowels & Consonants** — Classifies letters
- 🔢 **Digits** — Counts numeric characters
- ␣ **Spaces** — Counts whitespace
- ❗ **Special Characters** — Counts symbols
- 🖥️ **GUI Version** — Built with Java Swing

## 🛠️ Built With
- Language: **Java**
- GUI: **Java Swing**
- Input: **Scanner Class**
- IDE: **Visual Studio Code**
- JDK: **JDK 17 (LTS)**

## 📁 Project Files
- `KeyboardInputAnalyzer.java` — Console version
- `KeyboardInputAnalyzerGUI.java` — GUI version
- `Report.pdf` — Complete project report
